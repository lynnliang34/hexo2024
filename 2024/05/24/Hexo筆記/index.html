Hexo筆記

環境安裝

 

Hexo 指令

  1. 建立部落格：hexo init 網站名稱
  2. 開啟伺服器：hexo server
  3. 新增一篇部落格：hexo new 部落格標題
  4. 實際輸出網頁：hexo genrate
  5. 清除快取：hexo clean
  6. 部屬網站：hexo deploy
  7. 新增頁面：hexo new page 頁面名稱

更改表頭分頁

  1. 安裝 theme 內建樣板
  2. themes 資料夾下多 landscape 資料夾
  3. 點選 _config.yml
  4. 編輯

一鍵部屬

  1. 安裝hexo-deployer-git.
  2. 清空 _config.yml 的現有資料，並新增:
  1. 編輯替換 username、project
  2. 執行 hexo clean && hexo deploy

圖庫

  1. 在 source 資料夾下，新增 image 資料夾
  2. 圖檔存放於此
  3. Markdown 語法
